# 论文导读：Hybrid CNN-Mamba模型用于多尺度眼底图像增强
## 1. 研究背景与动机
眼底图像增强是医学影像辅助诊断的基础环节，传统CNN模型擅长局部纹理提取，但对长距离全局特征建模能力有限；Transformer参数量大，部署成本高。该论文结合CNN的局部优势与Mamba的高效序列建模能力，提出混合架构，解决多尺度眼底图像增强问题。
## 2. 核心方法
模型由CNN特征提取层、Mamba序列建模层、多尺度融合模块组成：
- CNN层：利用卷积捕捉眼底图像的血管、病灶等局部纹理特征。
- Mamba层：基于选择性状态空间模型，高效建模全局上下文特征，且参数量远小于Transformer。
- 融合模块：将不同尺度的局部与全局特征融合，生成增强后的图像。
## 3. 主要结果
在公开眼底图像数据集（如 IDRiD、DDR）上，该模型在峰值信噪比（PSNR）和结构相似性（SSIM）指标上显著优于纯CNN、纯Mamba及传统方法，实现了更高质量的图像增强。
## 4. 个人小结
该混合架构充分发挥了CNN与Mamba的互补优势，为医学影像增强提供了新思路。但模型在复杂病变图像上的泛化性仍需进一步验证，未来可引入轻量化设计降低部署门槛。
